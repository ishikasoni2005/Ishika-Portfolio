# 🌟 Ishika's Portfolio

A **modern, responsive, and professional developer portfolio** built with **React.js** and **TailwindCSS** to showcase skills, projects, and experience. Designed for aspiring software engineers aiming for top tech companies like FAANG.

---

## 🚀 Features

- **Responsive Design:** Mobile, tablet, and desktop ready  
- **Dark & Light Mode** toggle for modern UI  
- **Animated Components** with Framer Motion for smooth transitions  
- **Sectioned Layout:** Hero, About, Skills, Projects, Experience, Contact, Footer  
- **Project Showcase:** Hover effects, tech stack, GitHub & live demo links  
- **Contact Form:** Integrated with EmailJS  
- **Social Links:** GitHub, LinkedIn, Twitter  
- **Resume Download:** Easy access for recruiters  
- **SEO Optimized:** Meta tags, Open Graph tags, proper page titles  
- **Performance Optimized:** Lazy-loaded images, reusable components  

---

## 🛠 Tech Stack

- **Frontend:** React.js, Vite  
- **Styling:** TailwindCSS  
- **Animations:** Framer Motion  
- **Icons:** React Icons  
- **Backend for Contact Form:** EmailJS  
- **Deployment:** Vercel / Netlify

## ⚡ Installation

1. Clone the repo  

git clone https://github.com/ishikasoni2005/Ishika-portfolio.git

2. Navigate to project folder  
cd portfolio

3. Install dependencies  
npm install

4. Run locally  
npm run dev
